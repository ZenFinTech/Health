# 老年健康标准
+ 重要脏器的增龄性改变未导致功能异常，无重大疾病，相关高危因素控制在其年龄相适应的达标范围内，具有一定的抗病能力
+ 认知功能正常，能适应环境，处事积极乐观，自我满意评价好
+ 恰当处理家庭和社会人际关系，积极参与家庭和社会活动
+ 日常生活活动正常，生活自理或基本自理
+ 营养状况良好，体重适中，保持良好社会生活方式
